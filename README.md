# KD Tech Manifesto

> Tecnología con proposito. Ingeniería con conciencía. Sofrware para personas.

---

## 📜 ¿Qué es Manifesto KD Tech?

KD TECH Manifesto es un documento vivo que define los principios técnicos, 
éticos y humanos que guían la creación de software en KD Tech.

No es marketing.
No es postureo.
Es ingeniería responsable documentada.

---

## 🧑‍🦽‍➡️ Accesibilidad como pilar.

Esta proyecto incorpora:
- Lectura asistida.
- Alto contraste.
- Tokens de diseño adaptables.
- Preparación para daltonismo y baja vision.
- Navegación semántica real.

La accesibilidad no es un extra. Es un requisito.
